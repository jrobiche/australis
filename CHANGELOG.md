# Australis Changelog

## 0.2.0 (2026-02-24)

`Added` Telnet Client App

- Run telnet commands
- Navigate the file structure and launch executables
- View and edit memory values

`Changed` About Page

- Added application icon with name and version beneath it
- Moved different sections of information into cards with buttons for external links

`Changed` Apps Page

- Added a chevron to application cards

`Changed` Game Library App

- Redesigned the game details section
- Added ability to toggle if boxart is interactive

## 0.1.0 (2025-12-06)

`Added` Home Page

- Manage console configurations

`Added` About Page

- Page containing details about the application

`Added` Asset Manager App

- Set an asset image from a local file, the Xbox Catalog, or Xbox Unity
- Import game data from console
- Upload assets to console

`Added` Game Library App

- Card featuring interactive 3D cover art, a description, details, and screenshots for the game
- Launch the game on the console
- Manage artwork
- Import game data from console

`Added` System Information App

- Live temperature, memory (RAM) usage, and console information
- Information about DashLaunch configuration
- View achievements for the currently running game
- View, create, and delete screenshots for currently running game
- Information about the Nova plugin
- See currently signed in profile information
- Live thread information with ability to suspend or resume main thread
